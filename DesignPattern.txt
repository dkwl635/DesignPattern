싱글톤 Singleton 패턴
특정 클래스의 객체가 오직 한개만 존재하도록 보장하는 패턴
메니저 클래스처럼 한개의 객체만 생성하여 사용하는 경우 유용한 패턴

유한 상태 기계 Finite-State Machine FSM
한번에 오직 하나의 상태만을 가지게 되며, 현재 상태(Current State)란 임의의 
주어진 시간의 상태를 칭한다. 이러한 기계는 어떠한 사건(Event)에 의해 한 상태에서 다른 상태로 변화할 수 있으면,  이 를
전이(Transtition)이라 한다. 특정한 유한 오토마톤은 현재 상태로부터 가능한 전이 상태와, 이러한 전이를 유발하는 조건들의 집합으로서 정의 된다.

메모리 풀 
필요한 메모리를 미리 할당받아 사용하는 기법으로 메모리 단편화를 피할수 있다.
메모리의 할당과 해제가 잦은 경우에 효과적이다.

커맨드 패턴 Command Pattern
요청을 객체의 형태로 캡슐화하여 사용자가 보낸 요청을 나중에 이용할 수 있도록 매서드 이름, 매개변수 등 요청에
필요한 정보를 저장 또는 로깅, 취소 할 수 있게 하는 패턴이다.

옵저버 패턴 Observer Pattern
객체의 상태 변화를 관찰하는 관찰자들, 즉 옵저버들의 목록을 객체에 등록하여 상태 변화가 있을때마다 메서드 등을 통해 객체가
직접 목록의 각 옵저버에게 통지하도록하는 디자인 패턴이다.
